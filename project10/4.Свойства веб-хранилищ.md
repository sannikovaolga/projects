# Свойства веб-хранилищ

"+" объект сохраняется
"-" объект не сохраняется

| Действие                                                                         | Local Storage |  Session Storage|
|----------------------------------------------------------------------------------|---------------|-----------------|
| обновление страницы                                                              | +             | +               |
| дублирование вкладки браузера                                                    | +             | +               |
| восстановление страницы ( "Ctrl + Shift + T")                                    | +             | +               |
| закрытие и открытие браузера (перезапуск браузера)                               | +             | -               |
